Cisco Switch Image Extraction Guide

Goal: Extract the two Cisco switch images from the PDF and save them as PNG files in the correct location.

What We Need:

  • Source: Switches - Catalyst 9000_Art.pdf
  • Page 2: C9200L-24T-4G Front (24-port switch)
  • Page 3: C9200L-48P-4G Front (48-port switch)

Step-by-Step Extraction:

Step 1: Open the PDF

Open the file: C:\Users\thisi\Documents\Jonathan Apps\network-rack-designer\library\Switches - Catalyst 9000_Art.pdf

Use any PDF viewer (Adobe Reader, Chrome, Edge, etc.)

Step 2: Navigate to Page 2 (24-port switch)

Find the page with: C9200L-24T-4G Front

You should see a horizontal switch image with port numbers 1-24 and G1-G4

Step 3: Extract Page 2 Image

Method A - Screenshot:

  1. Zoom in on the switch image until it's clear and large
  2. Take a screenshot of just the switch (Windows: Win + Shift + S)
  3. Crop to include only the switch body (no text labels)
  4. Save as PNG

Method B - Right-click (if supported):

  1. Right-click on the switch image
  2. Select "Save image as..." or "Copy image"
  3. Save as PNG

Step 4: Save Page 2 Image

Save the 24-port switch image as:

C:\Users\thisi\Documents\Jonathan Apps\network-rack-designer\library\device-images\switches\cisco-c9200l-24t-4g.png

Step 5: Extract Page 3 Image (48-port switch)

Navigate to page 3 with: C9200L-48P-4G Front

This shows a switch with two rows of ports (1-24 bottom, 25-48 top) plus G1-G4

Use the same extraction method as Step 3

Step 6: Save Page 3 Image

Save the 48-port switch image as:

C:\Users\thisi\Documents\Jonathan Apps\network-rack-designer\library\device-images\switches\cisco-c9200l-48p-4g.png

Image Requirements:

  • Format: PNG (preferred) or JPG
  • Background: Transparent or white
  • Content: Switch front panel only (no text labels)
  • Quality: High resolution for clarity
  • Cropping: Tight crop around switch body

After Extraction:

Test the Images

  1. Open: simple-test.html
  2. Click "Test 24-Port Image" and "Test 48-Port Image"
  3. Images should load instead of showing "Image not found"
  4. If successful, the hybrid system is ready!
